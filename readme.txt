== Description ==

Makes bbPress "Notify me of follow-up replies via email" check box to be checked always unless user disables it. The user does not have to remember to tick the check box the recieve email replies everytime they post a topic or reply in the forum.

Yes, you can uncheck the Box, it is not permanent. Only enables checked by default each and everytime someone goes to create a topic or reply within the bbPress Forums.

== Installation ==

1. Upload the entire 'stay-checked-bbpress' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enjoy! ;)
